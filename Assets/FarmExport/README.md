# GOAP Project

Agents plant wood and cotton, then they harvest it, bring it to the barn.

When a truck will stop, one agent loads it, sending the resources elsewhere.

## Agents (check Agents folder):
* WoodFarmer
* CottonFarmer
* TruckLoader

## Actions (check actions folder):
* PlantTree
* PlantCotton

* CollectWood
* CollectCotton

* BarnDepositCotton
* BarnDepositWood

* BarnTakeCotton
* BarnTakeWood

* LoadTruck
